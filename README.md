## ⚡ Usage

**Requirements:** Hyprland, [Quickshell](https://quickshell.outfoxxed.me), `cliphist`, `playerctl`, `brightnessctl`, `nmcli`

```bash
# Clone the repo
git clone https://github.com/iamSt3el/Nebula.git ~/.config/quickshell

# Launch
quickshell -p ~/.config/quickshell
```

> [!NOTE]
> Make sure Quickshell is installed and your Hyprland session is running before launching.
